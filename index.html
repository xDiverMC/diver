FAQ

Pertanyaan Umum

Temukan jawaban dari pertanyaan yang sering ditanyakan pemain

🎮 Bagaimana cara join server Arvenia?
Untuk Java Edition: Buka Minecraft → Multiplayer → Add Server → masukkan IP play.arvenia.xyz → Join!

Untuk Bedrock Edition: Buka Minecraft → Play → Servers → Add Server → masukkan IP play.arvenia.xyz dengan Port 19132 → Join!
📱 Apakah bisa bermain dari HP (Bedrock)?
Tentu bisa! Arvenia mendukung crossplay antara Java & Bedrock. Kamu bisa bermain dari HP, Tablet, PC, bahkan Console. Semua pemain bermain di server yang sama!
💰 Bagaimana cara mendapatkan uang in-game?
Ada banyak cara! Kamu bisa mining & jual item ke server shop, menyelesaikan quest harian/mingguan, membuat player shop untuk jual ke pemain lain, memancing, atau farming. Semakin aktif bermain, semakin kaya! 💵
🏠 Bagaimana cara claim land?
Gunakan Oak Fence dan Oak sign untuk claim area! Klik kanan di dua titik diagonal untuk membuat claim. Kamu mendapatkan claim blocks gratis setiap jam bermain. Gunakan /rp addmember [nama] untuk memberikan akses ke temanmu.
🎰 Apa itu sistem Gacha?
Sistem Gacha adalah fitur unik di Arvenia dimana kamu bisa menggunakan Gacha Tokens untuk mendapatkan item acak! Mulai dari item biasa hingga item Mythical super rare. Gacha Tokens bisa didapat dari quest, event, atau beli di store.
🛒 Apakah pembelian di store aman?
100% aman! Kami menggunakan proses manual melalui WhatsApp admin terpercaya. Semua item akan langsung dikirim ke akun in-game kamu setelah pembayaran dikonfirmasi. Jika ada masalah, tim support kami siap membantu 24/7 melalui Discord!
🔒 Saya lupa password, bagaimana cara reset?
Buka Discord server Arvenia → channel #support → buat tiket baru → jelaskan bahwa kamu lupa password. Tim staff akan memverifikasi kepemilikan akun dan membantu reset password kamu.
⚡ Version Minecraft berapa yang didukung?
Arvenia mendukung Java Edition 1.17 - 1.21+ (rekomendasi versi terbaru untuk pengalaman terbaik) dan Bedrock Edition versi terbaru. Kami selalu update mengikuti versi terbaru Minecraft!
